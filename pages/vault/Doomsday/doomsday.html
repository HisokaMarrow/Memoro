Doomsday Algorithm

The Doomsday Algorithm is a mental method for calculating the day of the week for any given date — without a calendar. It’s fast, fun, and used by memory athletes around the world.

Step-by-Step Example: March 15, 2023

  1. Anchor the century: 2000s = Tuesday (2)
  2. Find the last two digits of the year: 23
  3. Add: 23 + floor(23/4) = 23 + 5 = 28
  4. Add month code for March: 3
  5. Add day of the month: 15
  6. Total: 2 (century) + 28 + 3 + 15 = 48
  7. Take mod 7: 48 % 7 = 6 → Saturday

Answer: March 15, 2023 falls on a Wednesday
